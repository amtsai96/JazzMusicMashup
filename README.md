# MusicSegmentSequencing
Music Segment Sequencing with Triplet Network
## Environment
- Python 3.7
- Pytorch 1.2.0
- Numpy 1.16.4
- Librosa 0.6.3
- Pydub 0.23.1
- madmom 0.16.1

You can replace the music segments in ```data``` with your propose audio data.<br>
Run the code
```python train.py```
to train the model
